📊 Account Executive KPI Scoring App (2024/25)

HOW TO USE:
1. Double-click the file: launch_kpi_app.bat
2. A browser window will open with sliders.
3. Fill in the scores, see final rating, and save result.
4. A file 'kpi_scores_log.csv' will be created automatically.

REQUIREMENTS:
✔️ Python must be installed (https://python.org)
✔️ Once only: install streamlit
   Open Command Prompt and type: pip install streamlit
